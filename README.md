# Ejecutar desarrollo

1. Clonar repositorio
2. Ejecutar
```
yarn install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la Base de Datos
```
docker-compose up -d
```


## Stack usado
* MongoDB
* NestJS
